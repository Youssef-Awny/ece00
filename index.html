    1-computer

    computer is programmable electronic device that accept data , present the results ,and can store the data or results

    Data VS Information
  • Data

    is a representation of a fact ,figure or idea. data can be a numer ,a word ,a picture ,or even a recording of sound

  • Information

    is a data that has been organized or presented in a meaningful fashion

    the language of computer

    computer uses combination of hardware and software to process data into information

  • hardware

    is any part of the computer that you can physically touch

  • software

    is a set of computer programs

    2-Input and output devices

    1-input device

  • Keyboard
  • Mouse
  • Microphone
  • Scanner
  • Bar code reader

    2-output devices

  • Monitor (is a peripheral device which display computer output on a screen)
  • Printer (is a peripheral device that produces a physical copy of thev computer’s output)
  • Plotter ( is a printer that uses a pen that moves over a large revolving sheet of paper)

    3- Motherboard and CPU

  • Motherboard

    It holds and allow communication between many of the electronic components of a system and provides connectors for other peripherals

  • Central processing unit ( CPU )

    it’s the brain of the computer which control all functions of the computer’s components

    4- Port

    A port is a place through which a peripheral device attaches to computer so that data can be exchanged between it and the operating system

      5- Memory vs. Storage

        memory

        Memory is a temporary workplace where the computer transfers the contents of a file while it is being used

          Storage

          Also known as mass media or auxiliary storage , refers to the various media on which a computer system can store data
